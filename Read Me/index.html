HTML Structure - top

The HTML sturcture for this plugin is as follows:



Code explanation:

[thumb_list_class]
This is the class name for the thumbs list.

[thumb_src]
This is the thumbnail src for the image/video. For YouTube videos use "http://i1.ytimg.com/vi/[video_id]/default.jpg", where [video_id] is the id of the YouTube video to embed.

[content_type]
This attribute sets the type of item to be rendered. For now there are two types - "img" and "youtube".

[content_src]
This attribute sets the content src. Fot images it is the src to the full size image, for YouTube videos it is the video ID.

[content_caption]
This attribute sets the caption text.

[img_alt_text]
This is the img alt attribute, and is not relevant to the plugins functionality.

How to find YouTube video ID:

To find Youtube videos ID, look at the URL of the video, and the ID should be a string of symbols after "watch?v="

If th URL looks like this - "http://www.youtube.com/watch?v=biFErW33YE0", then the ID is "biFErW33YE0".
